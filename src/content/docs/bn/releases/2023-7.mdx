## New Features

- Terraria has been added.
- Vanilla Bedrock will now auto-update
- Instead of Eggactyl relying on multiple files to know what you are running, it now relies on just 1 config file.
- The egg now has a flag in which you can disable the default `-Xmx` and `-Xms` flags with java programs. [Flag](/flags#current-flags)
- Lavalink now grabs the latest version always
- Switched to a new API.

## Updates

- Debian has been updated to Debian 12 "Bookworm"
- Switched from jabba to SDKMan
- Switched Java Adopt to Java Temurin
- "Other" category has been renamed to "Steam Game Servers"
- Removing NodeJS 10, 12, 15. Replacing with NodeJS 14, 16, 18, LTS
- Java Versions will now be: 8, 11, 17 to have builds that still get security updates.
- Changed Magma to use 1.12.2, 1.18.2, 1.19.3.
- MC proxies are updated to the latest version
- Egg will not be forced to restart if you give an invalid input or choose a selection in the install menu.
- You can now go back to the previous section when choosing a type of software.
- API now supports IPv6

## Removal

- Magma 1.16.5 has been removed since it has been archived by the developers.
- PowerNukkit has been removed.
- Mohist has been removed.
