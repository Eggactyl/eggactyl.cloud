---
title: Variabel Lingkungan
---

Ini adalah inventaris untuk semua variabel lingkungan baik untuk Eggactyl, atau untuk program Anda.

## Dari Pterodactyl
Karena Eggactyl berjalan di bawah Pterodactyl, variabel berikut diberikan ke aplikasi Anda.

- `SERVER_PORT` : Menunjukan port server utama.
- `SERVER_MEMORY` : Menunjukan jumlah memori yang teredia dalam MiB.

## Untuk Server Minecraft Java
- `JAVA_VERSION` : Menyetel versi Java untuk aplikasi Anda (`misal 17`)
- `JAVA_ARGUMENTS` : Menyediakan cara untuk menyampaikan argumen untuk JVM.

## Untuk Bot Discord
- `NODE_VERSION` : Menetapkan versi NodeJS untuk aplikasi Anda. (`misal 18`)
- `BOT_FILENAME` : Menyetel file default untuk dijalankan saat startup.

## Untuk Server Suara
- `QUERY_PORT` : Mengatur port kueri untuk TeamSpeak3/TeaSpeak.
- `FILE_PORT` : Mengatur port file untuk TeamSpeak3/TeaSpeak.
