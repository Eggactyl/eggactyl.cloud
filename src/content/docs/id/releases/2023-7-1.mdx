Ini adalah pembaruan perbaikan bug dengan ekstensi pada [2023.7](/releases/20237)

## Perbaikan bug
- Memperbaiki masalah daftar perangkat lunak yang tidak muncul jika ada file konfigurasi yang tidak lengkap
- Memperbaiki masalah dengan perintah java yang tidak ditemukan karena beberapa flag berubah di pengeksekusi.
- Memperbaiki masalah dengan MC java jika Anda memilih versi Java yang salah, sekarang Anda tidak dapat menginstal versi yang salah.
- Memperbaiki masalah ketika flag tidak diteruskan ke program itu sendiri.
