## Fitur baru
- Terraria telah ditambahkan.
- Vanilla Bedrock sekarang akan pembaruan otomatis.
- Daripada mengandalkan banyak file untuk mengetahui apa yang Anda jalankan, Eggactyl kini hanya mengandalkan 1 file konfigurasi.
- Egg sekarang memiliki flag di mana Anda dapat menonaktifkan flag `-Xmx` dan `-Xms` bawaan dengan Java. [Flag](/flags#current-flags)
- Lavalink sekarang akan menggunakan versi terbaru
- Beralih ke API baru.

## Updates
- Debian telah diperbarui ke Debian 12 "Bookworm"
- Beralih dari jabba ke SDKMan
- Beralih dari Java Adopt ke Java Temurin
- Kategori "Lainnya" telah diubah menjadi "Server Permainan Steam"
- Menghapus NodeJS 10, 12, 15. Menggantinya dengan NodeJS 14, 16, 18, LTS
- Versi Java sekarang adalah: 8, 11, 17 untuk versi yang masih mendapatkan perbaruan keamanan
- Mengubah Magma untuk menggunakan 1.12.2, 1.18.2, 1.19.3.
- Server proksi MC diperbarui ke versi terahkir
- Egg tidak akan dipaksa mulai ulang jika Anda memberikan masukan yang tidak valid atau membuat seleksi di menu instalasi
- Anda sekarang dapat kembali ke bagian sebelumnya ketika memilih jenis perangkat lunak.
- API sekarang mendukung IPv6

## Penghapusan
- Magma 1.16.5 telah dihapus karena telah diarsipkan oleh pengembang.
- PowerNukkit telah dihapus.
- Mohist telah dihapus.
