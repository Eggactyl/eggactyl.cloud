---
title: Flags
---

Eggactyl dispose de flags qui vous permettent de modifier le comportement de l'application.
## Flags actuels

- `--no-java-args` : N'inclut pas les flags `-Xmx` et `-Xms` lors de l'exécution d'une application Java.
- `--no-shell-check` : Désactive toute vérification des scripts shell pour les applications.
