## Quels logiciels puis-je installer avec Eggactyl ?

- Minecraft Java :
 - _[Vanilla](https://www.minecraft.net/), [Paper](https://papermc.io), [Purpur](https://purpurmc.org), [Magma](https://magmafoundation.org), [Spigot](https://www.spigotmc.org), [Forge](https://files.minecraftforge.net/net/minecraftforge/forge/), [Fabric](https://fabricmc.net), [Cuberite](https://cuberite.org)_
- Minecraft Bedrock :
 - _[Vanilla](https://www.minecraft.net/), [PocketMine-MP](https://pmmp.io)_
- Minecraft Proxies :
 - _[BungeeCord](https://www.spigotmc.org/wiki/bungeecord/), [WaterDog](https://waterdog.dev), [Waterfall](https://papermc.io/software/waterfall), [Velocity](https://papermc.io/software/velocity)_
- Bots Discord :
 - NodeJS, Python, PHP
- Serveurs de voix et de chat :
 - _[TeamSpeak3](https://www.teamspeak.com/), [Lavalink](https://lavalink.dev)_
- GTA :
 - _[SA\:MP](https://sa-mp.com)_
- Steam :
 - _Insurgency, Terraria_

## J'ai reçu un message disant que j'étais sur la liste noire pour "Essai gratuit"

Eggactyl a un système pour détecter les hébergeurs qui ont des essais gratuits et les met sur liste noire.

Si vous souhaitez mettre votre serveur sur liste blanche, vous pouvez contacter le support sur notre [Serveur Discord](https://discord.eggactyl.cloud)
