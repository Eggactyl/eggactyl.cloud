Il s'agit d'une mise à jour de correction de bogues avec une extension sur [2023.7](/fr/releases/20237)

## Correction de bugs

- Correction d'un problème avec la liste des logiciels qui ne s'affichait pas si le fichier de configuration était incomplet.
- Correction d'un problème avec la commande java qui n'était pas trouvée en raison de la modification de certains flags dans le runner.
- Correction d'un problème avec MC java si vous choisissez la mauvaise version de Java, maintenant il ne vous laissera pas installer les versions incorrectes.
- Correction d'un problème où le flag n'était pas transmis au programme lui-même.
