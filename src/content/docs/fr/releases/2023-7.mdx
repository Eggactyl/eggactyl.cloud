## Nouvelles fonctionnalités

- Ajout du jeu Terraria.
- Minecraft Bedrock vanilla se mettra à jour automatiquement.
- Plutôt que d'utiliser plusieurs fichiers pour savoir quel jeu est installé, nous utilisons maintenant un seul fichier.
- L'egg a maintenant un flag dans lequel vous pouvez désactiver les drapeaux par défaut `-Xmx` et `-Xms` avec les programmes Java. [Flags](/fr/flags#current-flags)
- Lavalink récupère désormais toujours la dernière version
- Passage à une nouvelle API

## Modifications

- Debian a été mis à jour vers Debian 12 "Bookworm"
- Passage de jabba à SDKMan
- Passage de Java Adopt à Java Temurin
- La catégorie "Other" a été renommée en "Steam Game Servers".
- Suppression de NodeJS 10, 12, 15. Remplacement par NodeJS 14, 16, 18, LTS
- Les versions de Java seront désormais : 8, 11, 17 pour avoir des versions qui reçoivent encore des mises à jour de sécurité.
- Modification de Magma pour utiliser les versions 1.12.2, 1.18.2, 1.19.3.
- Les proxies MC sont mis à jour avec la dernière version.
- L'Egg ne sera pas forcé de redémarrer si vous donnez une entrée invalide ou si vous choisissez une sélection dans le menu d'installation.
- Vous pouvez maintenant revenir à la section précédente pour choisir un type de logiciel.
- L'API prend désormais en charge IPv6

## Suppressions

- Magma 1.16.5 a été supprimé car il a été archivé par les développeurs.
- PowerNukkit a été supprimé.
- Mohist a été supprimé.
