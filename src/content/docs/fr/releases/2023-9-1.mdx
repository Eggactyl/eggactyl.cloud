## Corrections de bugs

- Correction d'un problème où l'egg ne récupérait pas les configurations par défaut pour MC Java
- Correction d'un problème où MC Java vanilla ne récupérait pas le bon fichier.
- Correction d'un problème où l'oeuf affichait "process not found" lors de l'arrêt du serveur.

## Mises à jour

- Ajout de la version 1.20.2 de MC Java :
- Vanilla
- Spigot
- Forge
- Fabric
- Paper
- Purpur
- Changement d'URL pour le CDN (l'ancienne URL sera fermée le 28 octobre 2023)
- Le CDN a maintenant des limites de taux plus strictes
- Ajout de l'email d'assistance aux messages d'assistance
- Ajout d'un message pour informer l'utilisateur que NPM est en cours de mise à jour
- Ajout d'un avis indiquant que NodeJS v14 sera supprimé dans une version ultérieure en raison de sa fin de vie et de l'entrée de NodeJS v20 dans la LTS à la fin du mois d'octobre.
- La v14 continuera à fonctionner pour tous les serveurs existants car elle récupère tout ce qui se trouve dans la configuration et l'envoie dans le NVM.
- Option ajoutée pour NodeJS v20
- SDKMan se met à jour automatiquement au démarrage.
